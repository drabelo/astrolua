# astrolua

A cute, romantic, modern astrology compatibility web app for couples.

## Run locally

Open `index.html` in your browser.

## Deploy with GitHub Pages

1. Push to `main` (this repo already has a Pages deploy workflow).
2. In GitHub, open **Settings → Pages** and set **Source** to **GitHub Actions**.
3. After the workflow completes, your app will be live at your Pages URL.
